# signal-handling-cpp
Examples from my presentation on signal handling in C++.

## How to compile and run

g++ -o no_handler 1_no_handler.cpp./no_handler

g++ -o custom_handler 2_custom_handler.cpp./custom_handler

g++ -o raise_example 3_raise_example.cpp
./raise_example

g++ -o kill_example 4_kill_example.cpp
./kill_example
